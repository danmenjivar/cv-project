# CV Maker

![](./screenshot.png)
CV Maker is an easy way to write a professional looking resume.

## Features

- Automatic formatting
- Save CV as PDF
- Export CV as JSON
- Load previously exported CV data
- Handles Personal Info, Education, Employment Experience, and Technical Experience
